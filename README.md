# ESP32 Метеостанция (учебный проект)

Простой учебный проект для начинающих, для сбора данных с датчиков AHT20 + BMP280 и вывода на OLED-дисплей.

## Что это?
- Учебный проект для освоения работы с I2C-устройствами
- ESP32 + датчики температуры/влажности/давления + OLED
- Базовый код без сложной логики

## Железо
- ESP32 DevKit
- Модуль AHT20 + BMP280 
- OLED 0.96" 128x64 (SSD1306)

## Подключение
- ESP32  -> Датчики + OLED
- 3.3V   -> VCC
- GND    -> GND
- GPIO21 -> SDA
- GPIO22 -> SCL

## Зависимости
- Adafruit AHTX0 Library
- Adafruit BMP280 Library
- Adafruit SSD1306

## Ключевые принципы работы:
- I2C шина - все устройства (дисплей, AHT20, BMP280) подключены к одним и тем же пинам SDA/SCL, но имеют разные адреса
- setup() - настраивает железо один раз при включении
- loop() - бесконечно повторяет: прочитать данные → показать на дисплее → вывести в Serial → подождать
- Non-blocking - код не использует сложные таймеры, просто ждет 2 секунды между измерениями

## Запуск
1. Собрать схему
2. Прошить код через Arduino IDE
3. Данные появятся на дисплее

## Что измеряет
- **Температура** (AHT20) - в градусах Цельсия
- **Влажность** (AHT20) - в процентах
- **Давление** (BMP280) - в гектоПаскалях
- **Высота** (BMP280) - расчетная высота над уровнем моря в метрах

## Особенности
- Давление конвертируется из Паскалей в гектоПаскали (/100.0)
- Высота рассчитывается относительно давления 1013.25 гПа
- OLED-дисплей обновляется каждые 2 секунды
